# Effective Work Breakdown Structure (WBS) Guidelines

Effective WBS guidelines:
- Use nouns, not verbs, to keep focus on deliverables.
- Employ a hierarchy to illustrate relationships among deliverables.
- Build the WBS collaboratively with the people who will produce the deliverables.
- Ensure each WBS leg contains unique deliverables; avoid duplication.
- Do not assume stakeholders know a deliverable is in scope; explicitly include it.
- Apply the 100% rule at every level: each level must contain 100% of the parent deliverable.
- There is no single correct way to organize the hierarchy; the final picture must make sense to all stakeholders.
