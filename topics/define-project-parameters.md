# Document Project Parameters in the PID

Project parameters include the total budget, a detailed budget breakdown, schedule milestones, the collaboration model with the client, and the first tangible target (e.g., MVP delivery). Clearly stating these constraints enables realistic planning and transparent stakeholder expectations.

## Can Answer

- How should I record the budget breakdown in a PID?
- What information about timelines belongs in a PID?
- Which collaboration details need to be captured in a PID?

## Examples

- Budget: $120,000 total – $70,000 development, $30,000 design, $20,000 QA.
- Timeline: 6‑week design sprint, 12‑week development phase, 2‑week testing period.

## Warnings

- Over‑promising on timeline or budget without supporting data can cause later bottlenecks.
