# Project Initiation Document (PID): Alternative Names and Origin

The PID originates from the PRINCE2 project management methodology, which defines a set of mandatory documents for project control. In other frameworks, similar artifacts are called a Project Charter (common in PMI/PMBOK) or a Project Brief. Regardless of terminology, the purpose remains the same: to capture the project’s purpose, scope, governance, and success metrics at the outset.
