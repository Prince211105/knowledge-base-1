# Alternative Names for the Project Initiation Document (PID) and Its PRINCE2 Connection

The Project Initiation Document is also known as a Project Charter or Project Brief in many methodologies. While the terminology varies, the purpose remains the same: to formally launch a project with defined scope, governance, and success criteria. The PID concept originates from the PRINCE2 methodology, which mandates a project initiation stage. However, most project management frameworks include a similar artifact to capture early‑stage project information.
