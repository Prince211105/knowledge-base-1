# Seven Core Sections of a Project Initiation Document (PID)

The seven steps are: 1) Provide the Context, 2) Define the Project Parameters, 3) Define the Specifics, 4) Define the Project Breakdown Structure and Resourcing Plan, 5) Define Who’s Who (team & RACI), 6) Identify Risks, Assumptions, Issues, and Dependencies, 7) Share the PID with stakeholders.
