# Primary Purpose of a Work Breakdown Structure (WBS)

The Work Breakdown Structure (WBS) is a higher‑level project artifact that supports the creation of schedules and budgets. It is a hierarchical decomposition of the total scope of work to be carried out by the project team to accomplish the project objectives and create the required deliverables. The WBS represents all of the deliverables to be produced by the project for which work activities will be defined, planned, and executed. It answers the question “What are we going to produce as a result of the project?” and provides the foundation for schedule, cost, and scope‑creep management. It does not contain activities, cost estimates, resource assignments, or dependency information.
