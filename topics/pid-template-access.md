# Download the Project Initiation Document (PID) Template

Members of the Digital Project Manager community have access to a comprehensive PID template. After becoming a member, you can download the template containing pre‑filled sections for project context, parameters, specifics, work breakdown structure (WBS), RACI matrix, risk register, and sign‑off fields. The template can be customized to match your project methodology.
