# How to download the Project Initiation Document (PID) template from The Digital Project Manager

The Digital Project Manager offers a comprehensive PID template for members. To access it:
1. Join the DPM membership community (free trial or paid tier).
2. Go to the “Resources” section and locate the “Project Initiation Document Template”.
3. Download the editable file (Word or Google Docs) and tailor it to your project.
Non‑members can request a sample by contacting the site or visiting the linked landing page.
