# Understanding the 100% Rule in Work Breakdown Structures

The 100% rule states that each decomposition level in a WBS must contain all deliverable elements that together represent 100 % of the parent element’s scope. When adding a new level, verify that every component of the parent is accounted for; missing items indicate a need for a higher‑level deliverable, while items that do not belong should be removed to keep the WBS within scope.
