# How to Keep the Project Initiation Document Current

Treat the PID as a living document; update sections when scope changes, new risks emerge, or team composition shifts. Record version changes and re‑circulate to keep all stakeholders aligned.

## Can Answer

- How often should I update the PID?
- What is the best way to track PID revisions?

## Examples

- Version 1.0 – initial sign‑off; Version 1.1 – added new risk of data privacy regulation.

## Warnings

- Neglecting updates can cause the PID to become outdated and lose credibility.
