# Digital Project Manager Podcast Mention on PID

## Overview

The article references a podcast episode where Ben Aston and Maik Stettner discuss using a PID to improve project kickoffs.

## Detailed Summary

The Digital Project Manager Podcast episode featuring Ben Aston and Maik Stettner explores how a Project Initiation Document (also called a project brief) can be used to align stakeholders at the start of a project and make kickoffs more effective. The conversation highlights the consequences of jumping into projects too quickly without proper documentation and underscores the PID’s role in preventing miscommunication. Listeners are encouraged to support the show via a membership link.

## Content

Sometimes projects go off the rails simply because we jump into them too fast.
Ben Aston talks to Maik Stettner about how we can use a project initiation document, or project brief to get everyone on the same page at the start of the project to make project kickoffs more effective.
