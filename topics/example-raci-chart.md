# Sample RACI Chart for a Project Initiation Document (PID)

Sample RACI Matrix:
| Deliverable / Activity | Project Manager | UX Designer | Developer | Client PM | Art Director |
|------------------------|------------------|------------|-----------|----------|--------------|
| Project Charter        | A                | C          | I         | R        | I            |
| UI Mockups             | C                | R/A        | I         | C        | I            |
| Functional Specs       | R                | C          | C         | A        | I            |
| Final Build            | I                | I          | R/A       | C        | C            |
| Client Acceptance      | C                | I          | I         | A/R      | I            |

**Key:** R – Responsible, A – Accountable, C – Consulted, I – Informed.
