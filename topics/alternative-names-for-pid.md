# Common Alternative Names for a Project Initiation Document (PID)

A Project Initiation Document (PID) is a formal record that captures essential project information at the outset, including purpose, objectives, scope, and governance. The PID is referred to by several alternative names depending on the methodology, the most common being Project Charter and Project Brief. While the terminology varies, the underlying intent of the document remains the same.
