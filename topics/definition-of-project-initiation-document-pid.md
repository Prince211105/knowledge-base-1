# What Is a Project Initiation Document (PID)?

A Project Initiation Document (PID) defines the project scope, management approach, and overall success criteria. It captures basic project information such as context, scope, team composition, and collaboration guidelines. The PID serves both as an internal guide for the project team and as a reference for external stakeholders.
